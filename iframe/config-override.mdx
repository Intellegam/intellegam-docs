# Configuration Override

Override your backend's default configuration by adding parameters to the IFrame URL. This lets you customize the chat interface for different use cases without changing your backend settings.

## How It Works

When you embed the chat IFrame, configuration is loaded in this order:

1. **Backend configuration** - Your backend's `/config` endpoint provides default settings
2. **URL parameters** - Any parameters you add to the IFrame URL override the backend defaults
3. **System defaults** - Used only if neither backend nor URL provides a value

Only parameters you explicitly include in the URL are overridden. All other settings remain as configured by your backend.

## Branding Parameters

Customize the visual identity of your chat interface.

| Parameter | Description | Example |
|-----------|-------------|---------|
| `title` | Chat window title displayed in the header | `title=Support%20Chat` |
| `titleLogo` | URL to a logo image shown in the header | `titleLogo=https://example.com/logo.png` |
| `chatLogo` | URL to a logo image shown in the chat area | `chatLogo=https://example.com/chat-icon.png` |
| `backgroundImg` | URL to a background image for the chat | `backgroundImg=https://example.com/bg.jpg` |

## Message Parameters

Customize the text content and prompts shown to users.

| Parameter | Description | Example |
|-----------|-------------|---------|
| `startMessage` | Initial greeting message shown when chat loads | `startMessage=How%20can%20I%20help%3F` |
| `startPrompts` | Suggested prompts shown as clickable buttons (JSON array) | `startPrompts=["Help%20me","Get%20started"]` |
| `inputPlaceholder` | Placeholder text in the message input field | `inputPlaceholder=Ask%20a%20question...` |
| `disclaimerMessage` | Disclaimer text shown below the chat | `disclaimerMessage=AI%20responses%20may%20vary` |

## Example

Here's a complete example with multiple overrides:

```
https://ai.intellegam.com/iframe?endpoint=https://api.example.com/chat&title=Acme%20Support&titleLogo=https://acme.com/logo.png&startMessage=Welcome%20to%20Acme%20Support!&startPrompts=["Track%20my%20order","Return%20an%20item","Contact%20support"]&inputPlaceholder=Type%20your%20question...
```

This creates a chat with:
- Custom title: "Acme Support"
- Company logo in the header
- Personalized welcome message
- Three suggested prompts for common tasks
- Custom input placeholder text

## Tips

- **URL encode values** - Spaces become `%20`, special characters need encoding
- **Use JSON for arrays** - `startPrompts` expects a JSON array format
- **Keep URLs short** - Browsers have URL length limits (~2000 characters)
- **Test your configuration** - Use the [IFrame Configurator](./configurator) to preview changes before deploying
