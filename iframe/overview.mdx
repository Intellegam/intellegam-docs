# IFrame Integration

Embed the Intellegam chat in your application as an IFrame with programmatic control and URL-based configuration.

## Core Capabilities

**[Host Control API](./host-control-api)** - Control the chat from your application using postMessage.

**[Query Parameters](./query-params)** - Configure backend endpoints and pass custom context through URL parameters.

## Basic Setup

```html
<iframe
  id="chat-iframe"
  src="https://ai.intellegam.com/iframe?endpoint=https://api.example.com/chat"
  width="400"
  height="600"
></iframe>
```

## Integration Patterns

### Static Embed

Load IFrame with query parameters. User interacts directly with the chat interface.

Configure using URL parameters - see [Query Parameters](./query-params) for details.

### Programmatic Control

Control the chat from your application using postMessage to send messages and attach files.

See [Host Control API](./host-control-api) for complete API reference.
